00043      * @brief Caches resources by object ID.
00044      * @details This cache effectively associates Active Resource with Core
00045      * Data. It bridges the difference between the Active Resource interface and
00046      * the Core Data interface. When you "find" active resources, all resource
00047      * attributes become available. However, when Core Data fetches managed
00048      * objects incrementally, the incremental-store interface expects only
00049      * object IDs at first. Subsequently, Core Data asks for attribute and
