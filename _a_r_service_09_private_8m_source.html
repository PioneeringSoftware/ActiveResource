00062 - (NSString *)defaultPrefixSource
00063 {
00064     // Asking for the site path answers only the non-base path as the path, even
00065     // if the URL comprises relative path components based on a base URL which
00066     // itself includes a path. The base URL's path disappears from the
00067     // equation. Is this a bug in Foundation? Work around it by computing the
